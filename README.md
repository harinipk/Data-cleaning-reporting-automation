Data Cleaning & Reporting Automation Project

Overview
This project focuses on automating the data cleaning, processing, and reporting workflow using Python. It takes a raw sales dataset, cleans it, performs analysis, and generates visual insights using charts.

The goal is to improve efficiency in data preprocessing and build an automated data analysis pipeline.

Objectives
- Automate data cleaning process
- Handle missing values and duplicates
- Convert inconsistent data into structured format
- Perform basic sales analysis
- Generate meaningful insights using visualization
- Save processed outputs for reporting

Tools and Technologies Used
- Python 3
- Pandas
- Matplotlib
- CSV Dataset

Dataset Information

The dataset contains sales-related information such as:
- Brand
- Model
- Product details
- Total Sales
- Other product attributes

Features of the Project

Data Loading
- Reads CSV dataset using Pandas

Data Cleaning
- Handles missing values
- Removes duplicate entries
- Converts text-based numeric values into proper numeric format

Data Analysis
- Calculates Total Sales
- Calculates Average Sales
- Identifies top performing brands

Data Visualization
- Bar chart for Top 10 Brands
- Histogram for Sales Distribution

 Automation Output
- Cleaned dataset saved automatically
- Charts saved as image files
- Summary report generated in terminal

Key Insights Generated
- Total sales value of dataset
- Average sales per entry
- Top selling brands
- Distribution pattern of sales data

## How to Run the Project

### Install required libraries
pip install pandas matplotlib

### Run the project
python automation.py

Output Generated
After execution, the project generates:
- Cleaned dataset (Cleaned_Sales_Data.csv)
- Bar chart (top_brands.png)
- Histogram (sales_distribution.png)
- Console summary report
